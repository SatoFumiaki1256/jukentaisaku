# 計算トレーニング機能 設計メモ

> ステータス: 🔲 未実装（外骨格のみ index.html に存在）
> 担当ファイル（予定）: `calc.html`

---

## 概要

製図試験で頻出の計算問題を繰り返し練習できるトレーニングツール。  
計算式の暗記だけでなく、数値を変えた問題を自動生成して本番に備える。

---

## 対象となる計算項目

| # | 計算項目 | 内容 |
|---|---|---|
| 1 | 建蔽率 | 建築面積 ÷ 敷地面積 × 100 |
| 2 | 容積率 | 延床面積 ÷ 敷地面積 × 100 |
| 3 | 道路斜線 | (道路幅員 + セットバック) × 勾配 |
| 4 | 北側斜線 | 高さ制限の計算（低層住居専用地域） |
| 5 | 採光計算 | 有効採光面積 = 開口面積 × 採光補正係数 |
| 6 | 面積換算 | ㎡ ↔ 畳 ↔ 坪 の変換 |
| 7 | 床面積計算 | グリッド（455mm）からの面積計算 |
| 8 | 公園緩和 | 施行令134条：対面幅員加算 |

---

## 問題形式（案）

### 穴埋め形式
```
敷地面積 180㎡、建築面積 108㎡ の場合、
建蔽率は [____] % である。
```

### 計算入力形式
```
敷地：幅 12m × 奥行 15m
道路幅員：6m（北側）
建蔽率 60%、容積率 200%

① 最大建築面積は？    [____] ㎡
② 最大延床面積は？    [____] ㎡
```

### 数値ランダム生成
- 毎回異なる数値で出題
- 難易度別（敷地形状・緩和条件なし/あり）

---

## データ構造（案）

```javascript
{
  id: "calc_001",
  type: "coverage",       // 計算タイプ
  difficulty: 1,
  template: {
    siteW: [8000, 10000, 12000, 15000],  // ランダム選択
    siteD: [10000, 12000, 15000, 18000],
    coverageRate: [50, 60, 70],
  },
  generate: (params) => ({
    question: `敷地 ${params.siteW/1000}m × ${params.siteD/1000}m、建蔽率 ${params.coverageRate}% の最大建築面積は？`,
    answer: (params.siteW/1000) * (params.siteD/1000) * params.coverageRate / 100,
    unit: '㎡'
  })
}
```

## UI（案）

1. 計算項目選択（カード一覧）
2. 問題表示 → 計算フォーム入力
3. 正誤判定（許容誤差: ±0.5㎡）
4. 解説表示（計算過程を表示）
5. スコア・正解率の記録

---

## 実装優先順位

1. 建蔽率・容積率の計算問題（最も基本）
2. 道路斜線計算
3. 面積換算（㎡・畳・坪）
4. ランダム数値生成
5. スコア記録（localStorage）

---

## 留意事項

- エスキスアプリの数値（455mmグリッド）と整合性を取る
- 計算式を必ず解説に含める
- 小数点の丸め方（四捨五入 or 切り捨て）を明記する
