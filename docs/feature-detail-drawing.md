# 詳細図トレーニング機能 設計メモ

> ステータス: 🔲 未実装（外骨格のみ index.html に存在）
> 担当ファイル（予定）: `detail.html`

---

## 概要

製図試験の「詳細図」セクションに特化した練習ツール。  
階段・基礎・軒先・窓回り等の詳細図を、  
お手本を見ながらトレース/描き方の要点を学べる。

---

## 背景・課題

二級建築士製図試験では平面図だけでなく **詳細図の描き方**が問われる。  
多くの独学者が「詳細図の練習方法がわからない」という悩みを持つ。

---

## 対象となる詳細図（案）

| # | 詳細図の種類 | 難易度 |
|---|---|---|
| 1 | 基礎断面図（布基礎・ベタ基礎） | ★★☆ |
| 2 | 軒先・破風の断面詳細 | ★★★ |
| 3 | 外壁断面図 | ★★☆ |
| 4 | 階段の断面・平面詳細 | ★★★ |
| 5 | 窓・サッシ回りの詳細 | ★★☆ |
| 6 | 床断面（フローリング・畳） | ★☆☆ |
| 7 | 柱・梁の仕口・継手 | ★★★ |

---

## 機能アイデア

### 1. お手本表示モード
- 詳細図のお手本画像（SVG or PNG）を表示
- 各部位の名称・寸法・材料をインタラクティブに表示
- タップで注釈を確認

### 2. 描き方チェックリスト
- 「軒先詳細を描く手順」のステップバイステップ
- 各手順のチェックボックスで進捗管理

### 3. クイズモード
- 「この部位の名前は？」「この寸法は何mm？」
- フラッシュカードと連携

---

## データ構造（案）

```json
{
  "id": "detail_001",
  "type": "foundation",
  "title": "布基礎断面図",
  "description": "木造2階建ての標準的な布基礎",
  "svgPath": "assets/detail_foundation.svg",
  "annotations": [
    { "id": "a1", "label": "捨てコンクリート", "x": 50, "y": 120 },
    { "id": "a2", "label": "フーチング幅 450mm", "x": 80, "y": 160 }
  ],
  "checklist": [
    "地盤面から基礎天端までの高さを記入（450mm以上）",
    "捨てコンクリートの厚さ（50mm）を記入",
    "フーチングの幅・高さを記入"
  ],
  "difficulty": 2
}
```

## UI（案）

1. **詳細図一覧**: 種類別タイル表示
2. **詳細図表示**: お手本SVG + 注釈（タップで表示/非表示）
3. **描き方ガイド**: ステップ形式のチェックリスト
4. **クイズモード**: 部位名・寸法の確認問題

---

## 留意事項

- 詳細図はSVG形式で作成（解像度に依存しない）
- 寸法は試験問題で実際に要求される寸法に基づく
- 木造2階建て（二級建築士試験の主要構造）に特化
