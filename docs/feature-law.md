# 法規コラム機能 設計メモ

> ステータス: 🔲 未実装（外骨格のみ index.html に存在）
> 担当ファイル（予定）: `law.html`

---

## 概要

製図試験に関連する建築基準法・施行令の条文を、  
「試験に出る箇所」に絞って分かりやすく解説するコラム機能。

独学者が「条文は難しくて読めない」という壁を越えるためのサポート。

---

## コンテンツ方針

- **試験に出ない条文は扱わない**（二級製図試験に特化）
- 条文の丸暗記ではなく「なぜそのルールがあるか」の理解を促す
- 図解・具体例を積極的に使う

## データ構造（案）

```json
{
  "id": "law_001",
  "law": "建築基準法 第28条",
  "title": "居室の採光",
  "summary": "住宅の居室には床面積の1/7以上の窓が必要",
  "body": "## 条文の要点\n...",
  "relatedCalc": "calc_005",
  "tags": ["採光", "居室", "必須知識"],
  "examFrequency": "高"
}
```

---

## UI（案）

1. **コラム一覧**: 優先度・タグ別フィルター
2. **コラム詳細**: 条文 → 解説 → 図解 → 計算例 → 関連コラム
3. **試験頻出度**: ★1〜3で表示
4. **関連機能リンク**: 「この法規を使った計算トレーニングへ」
