# 解き方Tips機能 設計メモ

> ステータス: 🔲 未実装（外骨格のみ index.html に存在）
> 担当ファイル（予定）: `tips.html`

---

## 概要

製図試験のエスキス・作図における「定石・コツ・注意点」を  
読み物形式（記事・ガイド）で提供するコンテンツ機能。

---

## ユーザーストーリー

- 「ゾーニングのやり方がわからない」→ 基本的な考え方を学ぶ
- 「吹き抜けを要求された時どうする？」→ パターンを知る
- 「採点で何点引かれるか怖い」→ 失点パターンを把握する

---

## コンテンツ構成案

### カテゴリ
| # | カテゴリ | 記事例 |
|---|---|---|
| 1 | エスキス基礎 | 課題文の読み方・優先順位のつけ方 |
| 2 | ゾーニング | パブリック/プライベートの分け方、日当たりのセオリー |
| 3 | 動線計画 | 家族動線・来客動線・サービス動線の分離 |
| 4 | 吹き抜け・特殊要素 | 吹き抜けの配置セオリー、ロフト・テラス |
| 5 | 敷地対応 | 隅切り・台形・北向き敷地への対応 |
| 6 | 失点ゼロ作戦 | よくある減点ポイントTOP10 |
| 7 | 時間管理 | 3時間30分の時間配分 |

---

## データ構造（案）

```json
{
  "id": "tip_001",
  "category": "エスキス基礎",
  "title": "課題文の読み方：3段階チェック法",
  "summary": "課題文を読む際に見落とさないための手順",
  "body": "## ステップ1: 敷地条件を確認\n...",
  "tags": ["エスキス", "基礎", "初心者"],
  "difficulty": 1,
  "readTime": 5
}
```

---

## UI（案）

1. **記事一覧**: カテゴリ別タブ + カード形式
2. **記事詳細**: マークダウンレンダリング、画像あり
3. **ブックマーク**: 気に入った記事を保存（localStorage）
4. **難易度フィルター**: 初心者 / 中級 / 上級

---

## 実装優先順位

1. `tips.json` に記事データ5〜10本を作成
2. `tips.html` の一覧 + 詳細表示UI
3. マークダウンレンダリング（marked.js 等）
4. ブックマーク機能

---

## 留意事項

- 図解（SVGやシンプルな画像）があると理解が早い
- エスキス練習ツールへの導線を随所に入れる
- 「これを読んだ後にすぐ練習できる」設計を意識する
